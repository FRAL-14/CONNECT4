How to install the game:

Once you have clicked on the Download button above, you will to see a .zip file in the download bar at the bottom left of your screen.

Once it finished downloading, you will find it in your file explorer. Now unzip the file in the same folder.

How to run:

The "path" is where your downloaded and extracted folder is. Find it and then copy the path.

  1. Open your terminal (Windows & Mac)
  2. cd Downloads
  3. javac -d IdkTheNameOfTheZipYet
  4. cd SameNameAsAboveWithout.zip
  5. java GameSession.java
